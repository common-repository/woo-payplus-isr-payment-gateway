**This is the PayPlus payment gateway for WooCommerce 3.0+**

== Description ==
תוסף תשלום לווקומרס באמצעות פיי פלוס.
The plugin allows you to use payplus payment gateway with the WooCommerce plugin.



== Installation ==
1. In your WordPress Dashboard go to \"Plugins\" -> \"Add Plugin\".
2. Search for \"payplus-payment-gateway-woocommerce\".
3. Install the plugin by pressing the \"Install\" button.
4. Activate the plugin by pressing the \"Activate\" button.
5. Open the settings page for WooCommerce and click the \"Checkout\" tab.
6. Click on the sub tab for \"PayPlus\".
7. Configure your PayPlus Gateway settings.
